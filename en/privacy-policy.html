5. Sharing with Third Parties

We do not share data except with consent, legal reasons, or to protect safety.

Dropbox API

Used for cloud integration; Dropbox's policy applies.

Firebase Analytics

Collects anonymized usage data to improve quality. Default OFF in EEA/UK and ON elsewhere. Users can change this anytime in Settings.

Firebase Crashlytics

Collects anonymous crash reports (stack traces, OS version) to improve stability. No personally identifying information is included.

Google Mobile Ads (AdMob)

Serves App Open Ads, banners, and interstitials. In EEA/UK we use Google UMP to obtain consent and show non-personalized ads (NPA) if consent is not given. On iOS we request ATT if required.

Firebase Remote Config

Delivers policy values like ad cooldowns or accessibility suppressions remotely. It does not store personal data; only numeric/boolean config values.

5.2 Consent and Defaults for Ads/Analytics

  • Analytics default: OFF in EEA/UK, ON elsewhere. Applied only on first run; afterwards we always respect the user‚Äôs in-app preference (Settings ‚Üí ‚ÄúAnonymous usage stats‚Äù).

  • Frequency caps: App Open/interstitial ads have cooldowns and session caps to avoid excessive display (tunable via Remote Config).
  • User choice: Users can change consent and analytics settings anytime in the app.

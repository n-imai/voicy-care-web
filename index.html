VoicyCareの特徴

🔊

音量200%増幅システム

通常の音楽アプリでは実現困難な200%音量増幅を実現。イヤホン使用時の安全な高音量再生で、聞こえにくい音楽も明瞭に。

  • イヤホン専用の強力増幅機能
  • 聴覚保護のための警告システム
  • 段階的な音量調整
👴

高齢者対応UI/UX

通常の150%サイズの大きなフォントとハイコントラスト配色で、見やすく操作しやすいインターフェースを実現。

  • 大きなタップ領域で操作しやすく
  • 直感的でわかりやすいアイコン
  • 高コントラストで見やすい配色
🎵

3つの音質プリセット

聴覚特性に合わせた「はっきり」「やわらかく」「静かに」の3つのプリセットで、最適な音質を簡単選択。

  • 「はっきり」- 人声・音声を明瞭に
  • 「やわらかく」- 耳に優しい柔らかい音質
  • 「静かに」- 騒音環境での聞き取り重視
☁️

マルチクラウド対応

Dropbox完全対応により大容量ファイル（最大4GB）のストリーミング再生を実現。将来的に他のクラウドサービスにも対応予定。

  • Dropbox完全統合（自動リフレッシュ対応）
  • 大容量ファイル（最大4GB）対応
  • バックグラウンドキャッシュ機能
⚡

高度な再生制御

ABリピート、プレイリスト管理、5バンドイコライザーなど、聴覚支援に最適化された高度な再生機能。

  • ABリピート機能（回数指定可能）
  • プレイリスト作成・管理
  • 5バンドイコライザー
🛡️

安全機能

聴覚保護を最優先に、音量増幅時の警告システムや長時間使用時のリマインダーで安全な使用をサポート。

  • 初回起動時の注意警告
  • 音量増幅時のポップアップ警告
  • 定期的な休憩リマインダー

こんな方におすすめ

👵

高齢者の方

聴覚の衰えを感じる方に、音楽を再び楽しんでいただくための専用設計。大きなボタンと明瞭な音質で、安心してお使いいただけます。

🦻

聴覚支援利用者

補聴器使用者の方にも最適化。音量増幅機能と音質調整で、補聴器と組み合わせてより快適な音楽体験を提供します。

👨‍⚕️

介護者・ご家族

高齢者をサポートする方にも使いやすい設計。シンプルな操作で、ご家族みんなで音楽を楽しむことができます。

🎓

語学学習者

スピード調整やABリピート機能で、語学学習にも最適。聞き取りにくい音声も明瞭に再生できます。

技術仕様

🛠️ 技術スタック

  • Framework: Flutter 3.0+
  • Language: Dart
  • Audio: just_audio + audioplayers
  • Architecture: Clean Architecture
  • State Management: BLoC Pattern

📱 対応プラットフォーム

  • iOS: 12.0以上
  • Android: API 21+ (5.0以上)
  • メモリ: 4GB以上推奨
  • ストレージ: 2GB以上の空き容量

🎧 対応ファイル形式

  • 音声: MP3, WAV, M4A, FLAC
  • 高音質: OGG, AAC
  • プレイリスト: JSON形式
  • 大容量: 最大4GBファイル対応

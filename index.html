VoicyCareの特徴

🔊

音量200%増幅システム

通常の音楽アプリでは実現困難な200%音量増幅を実現。イヤホン使用時の安全な高音量再生で、聞こえにくい音楽も明瞭に。

  • イヤホン専用の強力増幅機能
  • 聴覚保護のための警告システム
  • 段階的な音量調整
🎛️

見やすく操作しやすいUI/UX

通常の150%サイズの大きなフォントとハイコントラスト配色で、見やすく操作しやすいインターフェースを実現。

  • 大きなタップ領域で操作しやすく
  • 直感的でわかりやすいアイコン
  • 高コントラストで見やすい配色
🎵

音量・音質プリセット

5段階の音量プリセットと、シンプル/詳細を切り替えできる音質設定で、最適な聴こえを簡単に。 プリセット値を明記し、選びやすくしました。

  • 音量プリセット: やわらか(30%) / 標準(70%) / はっきり(100%) / くっきり(130%) / 力強く(150%)
  • 音質プリセット: やわらか(0.8倍) / 静か(0.9倍) / はっきり(1.2倍)
  • イコライザープリセット（クラシック/ボーカル/ロック/フラット）
☁️

クラウド連携

Dropboxに対応。大容量ファイル（最大4GB）のストリーミング再生を実現。

  • Dropbox完全統合（自動トークンリフレッシュ対応）
  • 大容量ファイル（最大4GB）対応
  • バックグラウンドキャッシュ機能
⚡

高度な再生制御

ABリピート、プレイリスト管理、5バンドイコライザーなど、聴覚支援に最適化された高度な再生機能。

  • ABリピート機能（回数指定可能）
  • プレイリスト作成・管理
  • 5バンドイコライザー
📢

広告と同意管理

音楽体験を最優先に、非侵襲的な広告運用と同意管理に対応。

  • バナー広告（非再生画面のみ）
  • インタースティシャル（クラウドストレージ入口で1日1回/セッション1回）
  • UMPによる同意管理（PA/NPA自動切替）
🛡️

安全機能

聴覚保護を最優先に、音量増幅時の警告システムや長時間使用時のリマインダーで安全な使用をサポート。

  • 初回起動時の注意警告
  • 音量増幅時のポップアップ警告
  • 定期的な休憩リマインダー

🛠️ 技術スタック

  • Framework: Flutter 3.16+
  • Language: Dart 3.2+
  • Architecture: Clean Architecture
  • State Management: BLoC Pattern
  • Audio Engine: just_audio + background + audioplayers
  • Ads: Google Mobile Ads SDK + UMP

📱 対応プラットフォーム

  • iOS: 15.0以上
  • メモリ: 4GB以上推奨
  • ストレージ: 2GB以上の空き容量

  • 音声: MP3, WAV, M4A, FLAC
  • 高音質: OGG, AAC
  • プレイリスト: JSON形式
  • 大容量: 最大4GBファイル対応

よくある質問

音量200%は安全ですか？

適切な音量でご利用ください。長時間の大音量は聴覚に影響する可能性があります。

クラウド連携なしでも使えますか？

端末内の音楽再生は可能です。クラウド再生にはDropbox連携が必要です。

対応ファイル形式は？

MP3/WAV/M4A/FLACなどに対応しています。

広告はどこに表示されますか？

再生画面以外に限定し、頻度は控えめに制御しています。

データはどこに保存されますか？

設定やトークンは端末内に保存。個人情報の外部保存は行いません。

未成年でも使えますか？

13歳未満の方は保護者の同意がある場合にご利用いただけます。

スクリーンショット

VoicyCareの特徴

🔊

音量200%増幅システム

通常の音楽アプリでは実現困難な200%音量増幅を実現。イヤホン使用時の安全な高音量再生で、聞こえにくい音楽も明瞭に。

  • イヤホン専用の強力増幅機能
  • 聴覚保護のための警告システム
  • 段階的な音量調整
🎛️

見やすく操作しやすいUI/UX

通常の150%サイズの大きなフォントとハイコントラスト配色で、見やすく操作しやすいインターフェースを実現。

  • 大きなタップ領域で操作しやすく
  • 直感的でわかりやすいアイコン
  • 高コントラストで見やすい配色
🎵

音量・音質プリセット

5段階の音量プリセットと、シンプル/詳細を切り替えできる音質設定で、最適な聴こえを簡単に。 具体的なプリセット値を明記し、選びやすくしました。

  • 音量プリセット: 30% / 70% / 100% / 130% / 150%
  • 音質プリセット: やわらか(0.8倍) / 静か(0.9倍) / はっきり(1.2倍)
  • イコライザープリセット（高音/低音/ボーカル/フラット）
☁️

マルチクラウド対応

Dropbox完全対応により大容量ファイル（最大4GB）のストリーミング再生を実現。将来的に他のクラウドサービスにも対応予定。

  • Dropbox完全統合（自動トークンリフレッシュ対応）
  • 大容量ファイル（最大4GB）対応
  • バックグラウンドキャッシュ機能
⚡

高度な再生制御

ABリピート、プレイリスト管理、5バンドイコライザーなど、聴覚支援に最適化された高度な再生機能。

  • ABリピート機能（回数指定可能）
  • プレイリスト作成・管理
  • 5バンドイコライザー
📢

広告と同意管理

音楽体験を最優先に、非侵襲的な広告運用と同意管理に対応。

  • バナー広告（非再生画面のみ）
  • インタースティシャル（Dropbox入口で1日1回/セッション1回）
  • UMPによる同意管理（PA/NPA自動切替）
🛡️

安全機能

聴覚保護を最優先に、音量増幅時の警告システムや長時間使用時のリマインダーで安全な使用をサポート。

  • 初回起動時の注意警告
  • 音量増幅時のポップアップ警告
  • 定期的な休憩リマインダー

👨‍⚕️

介護者・ご家族

🎓

語学学習者

スピード調整やABリピート機能で、語学学習にも最適。聞き取りにくい音声も明瞭に再生できます。

技術仕様

🛠️ 技術スタック

  • Framework: Flutter 3.16+
  • Language: Dart 3.2+
  • Audio: just_audio + just_audio_background + audioplayers
  • Architecture: Clean Architecture
  • State Management: BLoC Pattern
  • Ads/Consent: Google Mobile Ads SDK + UMP

📱 対応プラットフォーム

  • iOS: 12.0以上
  • Android: API 21+ (5.0以上)
  • メモリ: 4GB以上推奨
  • ストレージ: 2GB以上の空き容量

🎧 対応ファイル形式

  • 音声: MP3, WAV, M4A, FLAC
  • 高音質: OGG, AAC
  • プレイリスト: JSON形式
  • 大容量: 最大4GBファイル対応
